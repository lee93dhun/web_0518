3. 태그 style 속성 사용

0. 선택자 : 꾸미기 대상
1. 태그명{ 속성명 = 속성값;}
2. .클래스명{속성명 = 속성값;}
3. #id{속성명 = 속성값;}
4. *{ } : 모든 요소 꾸미기
5. 여러개 선택자 구분 : , 구분
- 태그명 , 태그명 { }
6. 하위 선택자 구분 : 공백 [띄어쓰기]
선택자 하위선택자{ }
